# learn_angular
follow Angular Tutorial

3 ways of use selector
- first use as a tag
```
selector: "app-string"
```
in html
```
<app-string></app-string>
```
- second use as a class name
```
selector: ".app-string"
```
in html
```
<div class="app-string"></div>
```
- third use as an atribut
```
selector: "[app-string]"
```
in html
```
<div app-string></div>
```
